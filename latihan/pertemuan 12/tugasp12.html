Belajar CSS Menggunakan Class

November, 2025

Class dalam CSS digunakan untuk memberikan gaya tertentu pada elemen-elemen HTML yang ingin memiliki tampilan seragam. Dengan menggunakan Class, kita bisa membuat halaman web yang lebih rapi, konsisten, dan mudah diatur tampilannya.

Misalnya, paragraf ini menggunakan class main agar teks lebih rapi dan rata kanan kiri. Class juga bisa digunakan untuk elemen lain seperti link, judul, maupun catatan tambahan.

Kunjungi situs W3Schools untuk belajar CSS lebih dalam dan mencoba contoh interaktifnya!
