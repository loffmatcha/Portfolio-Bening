Tentang CSS

Info: CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dokumen HTML.

CSS memungkinkan kita untuk memisahkan konten (HTML) dari presentasi (desain). Dengan CSS, kita dapat mengubah tampilan seluruh website hanya dengan mengubah satu file CSS.

Metode Embedding adalah salah satu cara penulisan CSS dimana kode CSS ditulis langsung di dalam tag <style> pada bagian <head> dokumen HTML. Metode ini cocok untuk styling yang hanya digunakan pada satu halaman tertentu.

Keuntungan Menggunakan CSS

Desain Menarik

Membuat website lebih menarik dengan berbagai efek visual dan animasi

Responsive

Website dapat menyesuaikan tampilan di berbagai ukuran layar

Cepat

Ukuran file lebih kecil dan loading lebih cepat

Mudah Maintenance

Perubahan style dapat dilakukan dengan mudah dan cepat

Perbandingan Metode CSS

Metode Kelebihan Kekurangan Penggunaan
Inline Cepat untuk testing Tidak efisien, sulit maintenance Style khusus satu elemen
Embedding Semua style dalam satu file Hanya untuk satu halaman Single page website
External Dapat digunakan banyak halaman Perlu file terpisah Multi-page website
